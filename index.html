Visual Logic

Visual Logic is a tool to represent existential graphs, a visual notation for logic expressions. Programmed as a console and web application in Unity and C#.

All logical expressions and operations are completly visual. At the moment Visual Logic can express any expression in propositional logic. Exisisential graphs are restricted to only four operations, dramatically reducing the number steps required to create a proof.

Developing as an independent study with Professor Bram van Heuveln
